## Installation
From your command line:

### Clone the repository
```bash
$ git clone git@github.com:HarryChopra/Slouchr.git
```

### Backend Setup
```bash
$ bundle install
$ rails db:create db:migrate db:seed
$ rails s
```

### Run the app
In your browser enter the url: http://localhost:3000/
